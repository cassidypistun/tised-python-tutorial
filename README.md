## What's in here

Three short notebooks to get you comfortable with Python and Jupyter before your first lab. You won't write any code — you'll run cells, change labelled values, and observe what happens.

| # | Notebook | What you'll do | Time |
|---|----------|---------------|------|
| 1 | **The Interface** | Run your first cells, trigger and fix a NameError, understand the kernel | ~15 min |
| 2 | **Variables & Values** | Change temperatures, city names, emission factors; toggle options with comments | ~20 min |
| 3 | **Libraries, Data & Charts** | Import libraries, read a data table, switch what a chart displays, run a full workflow | ~25 min |

---

## How to open the notebooks

1. Click the **Launch in Binder** button above
2. Wait for the environment to build (~1–2 min)
3. In the Jupyter Lab file panel on the left, double-click `Notebook_1_The_Interface.ipynb`
4. Press **Shift + Enter** to run each cell — that's all you need

In your SEAD courses, a TA will set up the full notebook. Your job is to run cells and change specific values — exactly what you practised here.
